# Brendan's Portfolio

Personal portfolio website built with HTML and CSS.

## Repository Structure

```
portfolio/
├── index.html
└── images/
    ├── logos/              # Company and organization logos
    ├── headshots/          # Photos
    └── case-studies/       # Project case study images, one folder per project
        ├── project-name-1/
        ├── project-name-2/
        └── ...
```

## Images

All images are hosted in this repository and referenced by URL in the HTML:

```
https://raw.githubusercontent.com/Brendan-s-Portfolio/portfolio/main/images/logos/example.png
```

This keeps the HTML file lightweight and easy to edit.

## Editing

To update the site, edit `index.html` directly on GitHub or clone the repo locally. To add images, upload them to the appropriate subfolder under `images/` and reference them using the raw GitHub URL format above.
